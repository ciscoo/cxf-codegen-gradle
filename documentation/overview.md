# Overview

CXF Codegen Gradle is a Gradle plugin for code generation with [Apache CXF](https://cxf.apache.org) from a single WSDL
document or several.

It is based on the [Maven plugin](https://cxf.apache.org/docs/maven-cxf-codegen-plugin-wsdl-to-java.html)
with similar configuration, but designed to work with Gradle's build model, conventions, and leans heavily on
Gradle idioms.

## Supported Gradle Versions

CXF Codegen Gradle supports Gradle 9.4 (or higher). Older versions of Gradle may work, but are not supported.
